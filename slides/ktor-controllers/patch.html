PATCH and PUT
  • PUT replaces entirely target resource
  • PATCH is a partial modification
  • No standard format of "partial modification"
  • RFC 7396 - JSON Merge Patch
  • PATCH needs to differentiate between provided null and missing field
  • API usually operates only on some subset of domain - DTOs needed
  • With DTOs we need mappings to domain objects
  • Delegates to the rescue!
