Coroutine Builders
  • kotlinx-coroutines provide several coroutine building functions
  • Coroutines can be lazy, meaning they are only executed when result is asked for
  • Builders accept context to run with
  • Builders defined on CoroutineScope merge scope's and passed contexts
  • Basic builders:
    • runBlocking
    • suspend withContext
    • suspend coroutineScope
    • CoroutineScope.async
    • CoroutineScope.launch
